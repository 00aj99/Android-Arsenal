数据来源国外网站 [android-arsenal](https://android-arsenal.com/) 的 APP，国外 Android 开发者比较喜欢逛的一个网站，实时更新各种开源项目、Libs。不过自带的广告有时候让人很不爽，尤其是在官方 APP 上。自己撸的没有广告，方便实时查看。
